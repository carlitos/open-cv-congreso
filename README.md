# open-cv-congreso


## Instalación de dependencias ⚙️

1. Instalación de `pipenv`:
    ```bash
    pip install pipenv
    ```

2. Crea un entorno virutal con `pipenv`, utilizando Python 3.8:
    ```bash
    pipenv shell --python 3.8
    ```

3. Instalación de dependencias:
    ```bash
    pipenv install -r requirements.txt
    ```
